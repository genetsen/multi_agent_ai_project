# Multi-Agent AI System for Media & Analytics Operations

## Project Goal
Design and implement a modular, multi-agent AI system that automates ingestion,
normalization, classification, validation, and project tracking across media plans,
first-party data, and reporting outputs.

## Agents
1. Data Harmonization Agent
2. Classification Agent
3. Quality Assurance Agent
4. Dashboard Validation Agent
5. Project Tracker Agent

## Cross-Agent Principles
- Explainability first
- Confidence scoring everywhere
- Human-in-the-loop by default
- No silent assumptions
- Composable and testable agents
